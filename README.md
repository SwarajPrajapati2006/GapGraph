# GapGraph
1. AI Resume Checker — Users upload or paste their resume, and the AI scores it across categories (clarity, keywords, formatting, impact), gives a letter grade, and suggests specific fixes. It can also tailor feedback for a target job role.
2. Graph / Analytics App — A dashboard that visualizes data (could be personal stats, GitHub activity, learning progress, finances) with interactive charts, filters, and trend lines — powered by real-time or user-uploaded data.
3. Skill Roadmap — An interactive learning path tracker where users pick a goal (e.g. "become a full-stack dev"), get a curated roadmap of skills/resources, and can mark progress as they go.
